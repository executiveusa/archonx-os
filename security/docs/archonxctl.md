# archonxctl (operator CLI)

`archonxctl` is the required operator tool for initializing and validating the ArchonX access-kernel stack.

## Standard flow
1. `archonxctl init`
2. `archonxctl install`
3. `archonxctl doctor`
4. `archonxctl e2e`

## Policy operations
- `archonxctl policy validate`
- `archonxctl policy push`

## Grants and audits
- `archonxctl grants request`
- `archonxctl grants approve`
- `archonxctl audit export`
